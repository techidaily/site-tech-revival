## Part 3\. FAQ about VOB Playback

###  1\. What is VOB?

What is VOB? VOB (Video Object) is a container format used for video and audio files, as well as subtitles, DVD menus, audio contained on DVDs. You can watch movies and videos in VOB format on Mac computers or Windows PC as long as you have a VOB file player. But this format is not widely accepted, and many media players are helpless to deal with it.

###  2\. How do I play VOB files on VLC Mac?

To apply VLC media player to play VOB files on macOS, check the below detailed instructions:

1. Install VLC and set it as the default media player.
2. Put your DVD into the media drive.
3. VLC should pick it up automatically. If VLC failed to auto detect and play DVDs, access to Video\_TS VOB folder and select open with VLC player.

###  3\. Can iMovie convert VOB to MP4?

Honestly, iMovie can't directly convert VOB to MP4\. iMovie doesn't accept VOB file format. You have to use a [Mac video converter](https://tools.techidaily.com/macxdvd/products/) to convert Video\_TS VOB file to MP4 or MOV format first, and then import the converted VOB file to iMovie for editing.
