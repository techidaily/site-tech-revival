## Part 1 How to Upgrade Old Mac/PC to 2017 New MacBook Pro

When you switch from old MacBook Air/Pro/iMac or Windows PC to 2017 new MacBook Pro, a key step to update MacBook Pro 2017 is to move all the important files and data to the new Mac. If you want everything exactly the same as the old Mac/PC, Migration Assistant or the data transfer tool for MacBook Pro 2017 update will be your best choice. 

1. Download and install Migration Assistant on 2017 MacBook Pro and old Mac/PC (Windows Migration Assistant for PC).
2. Connect the 2017 MacBook Pro and old Mac/PC to the same network (Wi-Fir or ethernet) and launch Migration Assistant on both devices.
3. On the new Mac, read the Instruction and click Continue > enter the admin password then requested > choose "From another Mac, PC, Time Machine backup or other disk" and then Continue.
4. On the old Mac or PC, choose "To another Mac" on the Migration Assistant window and enter the admin password when requested.
5. Shut down other programs to ensure proper old Mac/PC to 2017 MacBook Pro upgrade, then enter the passcode of Migration Assistant on new MacBook Pro to the old Mac/PC.
6. Select if you want to transfer user data, applications, and settings and then click Transfer.

### Part 2 How to Update MacBook Pro 2017 to macOS 10.13 

As Apple updates its operating systems frequently and is releasing macOS 10.13 the same time, there are also users wonder how to upgrade software on MacBook Pro 2017 to macOS 10.13 latest releases. To accomplish 2017 MacBook Pro [update to macOS 10.13](https://shop-links.co/link/?exclusive=1&publisher_slug=itechdaily19598&url=https%3A%2F%2Fwww.macxdvd.com%2Fmac%2Fhow-to-update-macos.htm), there are two steps you need to go: backup MacBook Pro data and download macOS 10.13\. You need Time Machine or the 2017 MacBook Pro update helper.

1. Connect an external hard drive to MacBook Pro 2017.
2. Click System Preferences> Time Machine> turn the bottom "Off" to "On."
3. Click the "Select Disk…" to select the drive or volume for Time Machine and it will start to backup before you update MacBook Pro 2017 to macOS 10.13.
4. After MacBook Pro data backup, open Apple Store and find macOS 10.13 update.
5. Click Download and enter Apple ID when requested.
6. After macOS 10.13 download, install the software update as per the instruction.
