###  Enable Tap to Click

 On macOS, you need to press the trackpad to perform a click. But worry not: you can make your Mac interpret taps as regular clicks instead. Go to System Settings > Trackpad, select the "Point & Click" tab, and enable "Tap to Click" so you can tap instead of clicking.

###  Bring Back the Right-Click

 Because early Macs used a single-button mouse, the OS is set up so that you need to hold the Control key while clicking to access the Mac equivalent of the Windows right-click menu. You can, however, enable the right-click in macOS.

 Go to System Settings > Trackpad, select the "Point & Click" tab, click the "Secondary Click" menu, and choose "Click or Tap with Two Fingers."

![macOS System Settings with the Secondary Click feature set to two-finger trackpad tap.](https://static1.howtogeekimages.com/wordpress/wp-content/uploads/2024/04/apple-macos-sonoma-system-settings-trackpad-point-and-click-section-secondary-click-menu-click-or-tap-with-two-fingers-selected.png)

###  Turn off Natural Scrolling

 When scrolling on macOS, content tracks your finger movements in the same way it does on the iPhone. This confuses the hell out of Windows users, but there's a remedy. In System Settings > Trackpad, select the "Scroll & Zoom" tab and switch off "Natural Scrolling."

![macOS System Settings with the Natural Scrolling feature disabled.](https://static1.howtogeekimages.com/wordpress/wp-content/uploads/2024/04/apple-macos-sonoma-system-settings-trackpad-scroll-and-zoom-section-natural-scrolling-disabled.png) 

 To scroll, drag two fingers on the trackpad, swipe vertically with one finger on the Magic Mouse, or turn the wheel on non-Apple mice.

###  Turn off Scrolling Inertia

 iPhone users are familiar with rubber band scrolling, and it's part of macOS too. It's yet another Apple peculiarity that often annoys Windows users. To disable the bouncing effect, and to be able to scroll with more control and precision, go to System Settings > Accessibility, click "Pointer Control," hit "Trackpad Options," and turn off "Use inertia when scrolling."

![macOS System Settings with the scrolling inertia feature disabled.](https://static1.howtogeekimages.com/wordpress/wp-content/uploads/2024/04/apple-macos-sonoma-system-settings-accessibility-motor-heading-pointer-control-trackpad-options-use-inertia-when-scrolling-disabled.png)

###  Prevent Mouse Pointer Shaking

 macOS lets you temporarily enlarge the pointer by quickly moving the mouse or your finger on the trackpad back and forth. It's useful if you've lost sight of it, but Windows lacks this option, so you may want to turn it off.

 Open System Settings > Accessibility > Display, and switch off the option "Shake mouse pointer to locate."

![macOS System Settings with the mouse shaking feature disabled.](https://static1.howtogeekimages.com/wordpress/wp-content/uploads/2024/04/apple-macos-sonoma-system-settings-accessibility-vision-heading-display-options-pointer-heading-shake-mouse-pointer-to-locate-disabled.png)

###  Change the Mouse Pointer Color

 The Windows pointer is white with a black outline, the opposite of macOS. To fix this, venture into System Settings > Accessibility > Display, then set "Pointer outline color" to black and "Pointer fill color" to white.

![Changing macOS pointer fill and outline colors in System Settings.](https://static1.howtogeekimages.com/wordpress/wp-content/uploads/2024/04/apple-macos-sonoma-system-settings-accessibility-vision-heading-display-options-pointer-outline-and-fill-colors.png) 

###  Disable Wallpaper Clicking

 By default, clicking your Mac's wallpaper hides all open windows to reveal your desktop widgets and volumes, but this utterly confuses folks who've never used a Mac.

 To disable this behavior, go to System Settings > Desktop & Dock, hit "Click wallpaper to show desktop items," choose "Only in Stage Manager," and turn off "Stage Manager."

![Disabling macOS wallpaper clicking and Stage Manager in System Settings.](https://static1.howtogeekimages.com/wordpress/wp-content/uploads/2024/04/apple-macos-sonoma-system-settings-desktop-and-dock-click-wallpaper-to-reveal-desktop-menu-only-in-stage-manager-selected-stage-manager-switch-disabled.png) 

##  Set Up Your Mac Keyboard Like Windows

 Muscle memory causing you problems? You can set up keyboard shortcuts and more on your Mac so they're similar to how you had them on Windows.

###  Turn on the Keyboard Navigation Feature

 On Windows, the Tab key selects buttons, menus, etc. You must enable two separate features on macOS to use your keyboard like a mouse.

 In System Settings > Keyboard, enable "Keyboard navigation." You can now press Tab and Shift+Tab to move focus between onscreen controls.

![macOS System Settings with the Keyboard Navigation feature enabled.](https://static1.howtogeekimages.com/wordpress/wp-content/uploads/2024/04/apple-macos-sonoma-system-settings-keyboard-navigation-enabled.png) 

 You can also enable additional keyboard navigation. In System Settings > Accessibility, click "Keyboard" and turn on "Full Keyboard Access. "Click "â“˜" to see the available navigation shortcuts. To display them when Full Keyboard Access is on, press Tab-H.

###  Configure Dictation

 To replicate the PC shortcut Windows+H for entering text by speaking, open System Settings > Keyboard and turn on the switch next to the microphone icon.

###  Enable the Function Keys

 Pressing function keys on a Mac doesn't invoke standard actions like on Windows. To change it, open System Settings > Keyboard > Keyboard Shortcuts, select "Function Keys," and enable "Use F1, F2, etc. keys as standard function keys."

![macOS System Settings with the option to use function keys as standard F1, F2, etc. keys enabled.](https://static1.howtogeekimages.com/wordpress/wp-content/uploads/2024/04/apple-macos-sonoma-system-settings-keyboard-shortcuts-options-function-keys-sidebar-use-f1-f2-as-standard-function-keys-enabled.png) 

###  Set the Emoji Panel Shortcut

 On PCs, pressing Windows+. (period) brings up the emoji panel. Macs use the Fn+E (or Globe+E) shortcut, but you can optimize it. Go to System Settings > Keyboard, click "Press fn key to" or "Press Globe key to," and choose "Show Emoji & Symbols."
